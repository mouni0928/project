# Sales Data Analysis Dashboard (Power BI)

## Project Overview
This project focuses on analyzing sales data to identify key business insights using Power BI. The dashboard provides a clear visualization of sales performance, product performance, and sales trends over time.

## Objectives
- Identify Top 5 products based on Sales, Quantity, and Profit
- Analyze sales trends over time
- Understand product performance and profitability
- Provide insights to support business decision making

## Tools & Technologies
- Power BI
- Data Modeling
- DAX (Data Analysis Expressions)
- Excel (Dataset)

## Key Features
- Top 5 products by Sales
- Top 5 products by Quantity
- Top 5 products by Profit
- Sales trends over time
- Interactive dashboard with visualizations

## Data Modeling
Implemented relationships between tables to create an efficient data model for analysis.

## DAX Measures
Created DAX measures to calculate:
- Total Sales
- Total Profit
- Total Quantity
- Product ranking

## Files Included
- sales_dashboard.pbix – Power BI dashboard
- sales_dataset.xlsx – Dataset used for analysis
